# Assets

Place your app icons and splash images here:

- icon.png          — 1024×1024 app icon
- splash.png        — 1284×2778 splash screen  
- adaptive-icon.png — 1024×1024 Android adaptive icon foreground
- favicon.png       — 48×48 web favicon

You can generate all sizes from a single image using:
https://www.appicon.co

Or run: npx expo-optimize (after installing)

For development, Expo provides default placeholder icons automatically.
